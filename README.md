# Explosive Chess

A Python project with a client-server-client architecture and pygame support for graphical clients.

## Structure
- `server/` - Server code
- `client1/` - First client (pygame)
- `client2/` - Second client (pygame)

## Requirements
- Python 3.x
- pygame (for clients)

## Getting Started
1. Install dependencies:
   - For server: `pip install -r server/requirements.txt`
   - For clients: `pip install -r client1/requirements.txt` and `pip install -r client2/requirements.txt`
2. Run the server:
   - `python server/server.py`
3. Run the clients:
   - `python client1/client.py`
   - `python client2/client.py`

## Notes
- This is a starter template. Implement your game logic in the respective files.
